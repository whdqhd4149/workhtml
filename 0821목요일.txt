순서7] html언어  
📌 HTML이란?
HTML (HyperText Markup Language)
→ 웹 페이지를 만들 때 사용하는 마크업 언어입니다.
→ 프로그래밍 언어가 아니라, 웹 브라우저가 문서를 어떻게 구조화하고 보여줄지를 정의하는 언어입니다.

📌 웹 크롤링이란?
웹 크롤링은 웹 사이트에서 자동으로 데이터를 수집하는 작업입니다.
보통 크롤러(Crawler) 또는 스파이더(Spider)라고 불리는 프로그램이 웹 페이지를 방문해서
HTML 문서를 가져오고, 필요한 정보를 추출(parse)해서, 저장(데이터베이스, 파일 등) 하는 과정입니다.

📌 웹 크롤링과 스크래핑
웹 크롤링 (Crawling) : 여러 웹 페이지를 자동으로 탐색하며 정보를 가져오는 과정
웹 스크래핑 (Scraping) : 특정 웹 페이지에서 원하는 데이터만 추출하는 작업
👉 두 개념은 구분해서 쓰기도 하지만, 보통 혼용됩니다.
